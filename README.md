# Digital Agency Website

This is a modern digital agency website built with React, Tailwind CSS, and Framer Motion.

## Project Structure

The project code is located in the `frontend` directory.

- `frontend/`: Contains the React application source code.
- `frontend/src/`: Component and page files.
- `frontend/public/`: Static assets.

## Getting Started

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up environment variables:
    - Create a `.env` file in the `frontend` directory.
    - Copy the contents from `.env.example`.
    - Fill in your EmailJS credentials.

4.  Start the development server:
    ```bash
    npm run dev
    ```

## Features

- **Modern UI**: Glassmorphism, particles background, and smooth animations.
- **Responsive Design**: Fully responsive layout for all devices.
- **Contact Form**: Functional contact form integrated with EmailJS.
- **Project Showcase**: Portfolio section with carousel.

## Technologies

- React
- Tailwind CSS
